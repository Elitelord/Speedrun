# MCAT Seed Deck

This is a small demo flashcard deck of ~60 introductory, MCAT-relevant
front/back Q&A cards, split into three sections stored as tab-separated
(`front<TAB>back`) files: `biobiochem.tsv`, `chemphys.tsv`, and
`psychsoc.tsv`. Each section maps to a tag — Bio/Biochem cards use
`mcat::biobiochem`, Chem/Phys cards use `mcat::chemphys`, and Psych/Soc cards
use `mcat::psychsoc` — so cards can be filtered and organized by MCAT content
area once imported. Facts are drawn from openly-licensed, commercial-safe
sources (OpenStax CC BY 4.0 titles and CC BY-SA Wikipedia) and rewritten in
original wording; see `ATTRIBUTION.md` for full source credits.
