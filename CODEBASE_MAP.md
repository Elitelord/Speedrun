# Codebase Map — Read This First

This repository is large. Before grepping or reading code broadly to find where
something lives, **consult the codebase map**:

➡️ **[`codebase-map/README.md`](./codebase-map/README.md)** — the table of contents.

It indexes the whole repo by area (Rust core, Python lib, Qt GUI, web frontend,
protobuf/IPC, translations, build system, docs) and points directly to the
relevant directories and files. Use it to navigate quickly, then read only the
specific files you need.

When you add or move code in a meaningful way, update the matching file in
`codebase-map/` so the index stays accurate.
